Projects

Real Time Syringe Pump

Android application to determine how much liquid the syringe outputs using a pump powered by a stepper motor. The application will communicate with a Bluetooth module attached to the SJ One board which will control the stepper motor driver to determine how many steps the stepper motor will take on the syringe pump.

Software: Android Studio, Eclipse

Languages: C/C++, Java

Car warning system that warns drivers of potential accidents by detecting red traffic lights and calculating safe braking distances. Car Vision is implemented on a Linux environment using OpenCV with C++ and interfaced to a microcontroller with a forward-facing camera.
